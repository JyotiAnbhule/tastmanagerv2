Task Management Service
A full-stack Task Management application built with Spring Boot 3 and Thymeleaf, featuring a modern UI powered by Tailwind CSS. This service allows users to manage projects and track tasks through their lifecycle (To-Do, In-Progress, Done).

Access the App: Open your browser and navigate to http://localhost:8080.

Registration: Go to /register to create a new account.

Login: Use your newly created credentials to access the dashboard.
